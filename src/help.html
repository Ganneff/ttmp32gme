Getting Started

1. Add mp3 files

Add one or more mp3 files on the "Upload" page. Only add one album at a time.

You can also add a picture which will be used as the album cover picture.

2. Configure and create gme files

On the "Library" page, you can configure and create gme files. Mp3 tag data of recently uploaded files will automatically be used to pre-populate the artist, album title and track info.

3. Print the control page(s)

Once you choose to print one or more gme files from the library, a new page will open that displays the albums and their tracks from the gme files that you selected for printing.

4. Copy the gme files onto the tiptoi pen

Connect the tiptoi pen to your computer and upload the gme files. (You can find the path to the gme files when you edit an album on the library page.)
